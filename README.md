## 第 1️⃣ 步：注册 GitHub 账号（3 分钟）

> 💡 GitHub 是全球最大的程序员社区，完全免费注册，跟你注册邮箱一样简单。

1. 用浏览器打开 👉 **https://github.com**
2. 右上角点 **Sign up**（注册）
3. 按提示填写：
   - **Email**：你的邮箱
   - **Password**：设个密码
   - **Username**：一个英文用户名（比如 `zhangsan2026`，以后网址里会用到）
4. 去邮箱收验证码，点验证链接
5. ✅ 注册完成

> 已经有 GitHub 账号的同学，直接跳到第 2 步。

---

## 第 2️⃣ 步：Fork（复制）项目到你账号（30 秒）

> 💡 **Fork** = 把别人的项目完整复制一份到你自己的账号下。复制后这份代码就是**你的了**，想怎么改怎么改。

1. 打开项目主页 👉 **https://github.com/weichenli0909-crypto/stock-report**
2. 确认右上角**已经登录**自己的账号
3. 点击右上角的 **Fork** 按钮（叉子图标 🍴）
4. 在弹出页面**直接点绿色的 Create fork** 按钮（其他默认即可）
5. 等 2-3 秒，页面会自动跳到**你自己账号下**的项目副本
6. ✅ 完成

**怎么确认成功了？** 看浏览器地址栏，网址应该变成：
```
https://github.com/你的用户名/stock-report
```
左上角的仓库名前缀也变成了你的用户名，下面还会显示一行小字 "forked from weichenli0909-crypto/stock-report"。

## 第 3️⃣ 步：启用 Actions（自动运行器）（30 秒）

> 💡 **GitHub Actions** = GitHub 免费送的"自动执行机器人"，可以定时帮你跑程序。

1. 在**你自己的**仓库页面（注意不是原作者的），点击顶部的 **Actions** 标签
2. 可能会看到一个黄色提示："Workflows aren't being run on this forked repository"
   👉 点绿色按钮 **I understand my workflows, go ahead and enable them**
3. 左侧列表里会出现 **📊 自动更新股票报告**，点击它
4. 右侧上方会有一个蓝色按钮 **Enable workflow**（启用工作流），点一下
5. ✅ 完成

**从现在起**，每个工作日（周一~周五）**北京时间 15:35**（A 股收盘后 5 分钟），GitHub 的云端服务器会自动：
- 采集当天所有股票的行情和资金数据
- 抓新闻做舆情分析
- 跑机器学习模型做预测
- 生成并更新你的报告网页

你什么都不用做，也不需要自己的电脑开机。

## 第 4️⃣ 步：启用 Pages（生成网页）（30 秒）

> 💡 **GitHub Pages** = GitHub 免费送的"网页托管"，让你的报告可以通过一个链接在任何地方打开。

1. 在你的仓库页面，点击顶部的 **Settings**（设置）标签
2. 在左侧菜单找到 **Pages**，点进去
3. 在 "Build and deployment" 这一栏：
   - **Source**：选 `Deploy from a branch`
   - **Branch**：选 `gh-pages`，右边的文件夹选 `/ (root)`
4. 点 **Save**（保存）
5. ✅ 完成

**你的专属报告网址会是**：
```
https://你的用户名.github.io/stock-report/
```

> ⏰ **第一次可能要等 2-5 分钟**才能打开。如果打开是 404，多刷新几次或等一会。

## 🍎 Mac 用户教程

### 第 1 步：下载代码

**方法 A：直接下载 ZIP（最简单，零基础首选）**

1. 打开 👉 **https://github.com/weichenli0909-crypto/stock-report**
2. 点绿色 **Code** 按钮 → **Download ZIP**
3. 双击下载的 zip 解压
4. 把解压后的 `stock-report-gh-pages` 文件夹**移到桌面**

**方法 B：用 Git 克隆（方便以后 `git pull` 一键更新）**

```bash
cd ~/Desktop
git clone -b gh-pages https://github.com/weichenli0909-crypto/stock-report.git
```

### 第 2 步：一键安装环境（3-5 分钟）

1. 打开刚才的文件夹
2. **右键**点击 `一键安装-Mac.command` → 选 **打开**
   > ⚠️ 第一次**一定要右键打开**，不能双击。因为 Mac 会提示"无法验证开发者"。
   > 右键打开一次之后，以后就可以双击了。
3. 弹出终端窗口后按 **Enter** 开始
4. 脚本会**自动**帮你装：
   - Homebrew（Mac 包管理器）
   - Python 3
   - 所有依赖库（akshare、pandas、matplotlib 等）
5. 过程中可能**要求输入开机密码**（用于安装 Homebrew），**密码输入时看不到字符是正常的**，直接输入后按 Enter
6. 看到 `🎉 安装完成！` 就好了

### 第 3 步：一键启动（以后每次就只点这个）

1. **双击** `启动股票报告-Mac.command`
2. 终端问 "是否要先更新最新股票数据？"，**输入 1 按 Enter**（第一次建议选 1）
3. 等 1-2 分钟，浏览器会**自动打开** http://localhost:8899
4. ✅ 搞定！

**以后每次使用**，只要双击 `启动股票报告-Mac.command` 就行。

> 💡 报告页面右上角可以：
> - 点 **⚡ 快速刷新**：只重新采集最新行情（20 秒）
> - 点 **🔄 完整刷新**：重新跑全部流程（2 分钟）
> - 切换 **☀️ 亮色 / 🌙 暗色** 主题

# 🎨 进阶：自定义关注的股票

不想只看我默认配置的股票？**完全可以自己改**！

## 在 GitHub 上改（云端版）

1. 打开你的仓库页面 → 点 `config.py` 文件
2. 点右上角 ✏️ 铅笔图标（编辑）
3. 找到 `STOCK_GROUPS`：
   ```python
   STOCK_GROUPS = {
       "AI算力": {
           "002230": "科大讯飞",
           "300059": "东方财富",
           # 👇 在这里加你想关注的股票
           "600519": "贵州茅台",
       },
       # 你也可以加一整个新板块
       "新能源": {
           "300750": "宁德时代",
           "002594": "比亚迪",
       },
   }
   ```
4. 拉到页面底部 → 点绿色的 **Commit changes**
5. 下一次自动运行就会用新的股票列表

## 在本地改（本地版）

用记事本 / TextEdit / VS Code 打开 `config.py`，改完保存，重新双击启动脚本即可。

> 💡 **股票代码**就是你炒股 App 里看到的 6 位数字（比如 `000001` 是平安银行、`600519` 是贵州茅台）。

完全可以！方式 A 全程只要点鼠标，不写一行代码。方式 B 的一键脚本会自动把环境装好，你不用管底层在干嘛。
</details>

<details>
<summary><b>Q：Actions 运行失败怎么办？（红色 ❌）</b></summary>

1. 点进失败的那条运行记录
2. 看看最上面写了什么错误
3. **最常见的两种原因**：
   - **数据源偶尔抽风**：等 2-3 小时再手动 Run workflow 一次，通常就好
   - **代码需要更新**：去仓库主页点 **Sync fork** → **Update branch**，然后重跑
4. 连续失败 > 2 天 → 去原仓库 Issues 反馈
</details>

<details>
<summary><b>Q：网页打开是 404 "There isn't a GitHub Pages site here"</b></summary>

- 检查 **Settings → Pages** 设置是否选了 `gh-pages` 分支（不是 main！）
- 检查 Actions 是否**至少成功运行过一次**（否则 gh-pages 分支是空的）
- 保存设置后等 3-5 分钟再打开
- 确认地址格式：`https://你的用户名.github.io/stock-report/`（最后的斜杠别漏）
</details>

<details>
<summary><b>Q：Mac 双击 .command 提示"无法验证开发者"</b></summary>

不要双击！改用**右键**（或按住 Control 点击）→ **打开** → 弹窗里再次点 **打开**。
只需要这样做**第一次**，以后就可以双击了。
</details>

<details>
<summary><b>Q：Windows 双击 .bat 一闪就关？</b></summary>

说明 Python 没装好（或者没勾 "Add Python to PATH"）。
**解决方法**：
1. 去控制面板卸载 Python（如果装过）
2. 重新双击 `一键安装-Windows.bat`，按提示下载 Python
3. 安装 Python 时**一定要勾选 "Add Python to PATH"**
</details>

<details>
<summary><b>Q：端口 8899 被占用？</b></summary>

启动脚本会自动关掉占用的老进程。如果还是不行，重启一下电脑最干净。
或者在脚本里把 `8899` 改成其他端口（如 `8900`）。
</details>

<details>
<summary><b>Q：AI 评分是什么意思？能跟着它买股票吗？</b></summary>

- **评分是相对排名**，不是涨跌概率。它告诉你"在关注池里哪些相对强"，不告诉你"明天会涨多少"。
- 80+ 强烈看好 / 65-80 看好 / 55-65 微看好 / 45-55 中性 / 35-45 微看空 / 20-35 看空 / <20 强烈看空
- **这只是辅助决策工具，不构成投资建议。** 实测近 155 天方向准确率 ≈ 52%，略好于抛硬币。
- 买卖请自己决策，盈亏自负。
</details>

<details>
<summary><b>Q：为什么周末 / 节假日看不到更新？</b></summary>

A 股周末和节假日不开盘，所以定时任务只在**周一~周五**运行。报告显示的是最近一个交易日的数据，正常。
</details>

<details>
<summary><b>Q：报告里的数据是从哪来的？安全吗？</b></summary>

- 数据来源：akshare（国内开源金融数据库）、yfinance（雅虎财经美股）、新浪财经等**公开接口**。
- 不需要你的股票账户、不需要任何个人信息。
- 网页上的报告是**只读**的，即使公开也只有股票分析，**没有任何隐私**。
</details>

<details>
<summary><b>Q：能不能让它跑得更频繁？比如每小时一次？</b></summary>

能，但不建议：
- 修改 `.github/workflows/update-report.yml` 里的 `cron` 表达式
- 例如 `'0 */1 * * *'` = 每小时整点跑一次
- ⚠️ 跑太勤会吃掉你的 Actions 免费额度，而且收盘前跑的都是盘中不完整数据
</details>

<details>
<summary><b>Q：我想让别人也能改、多人协作怎么办？</b></summary>

在你的仓库 **Settings → Collaborators** 里添加他们的 GitHub 用户名即可，对方接受邀请后就能一起编辑。
</details>
