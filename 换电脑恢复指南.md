### 方法二：手动步骤（进阶用户）

#### ① 先装 Git（如果没装）

- **Mac**: 打开终端输入 `git --version`，系统会提示安装
- **Windows**: 去 https://git-scm.com/download/win 下载安装

#### ② 克隆仓库

```bash
cd ~/Desktop
git clone https://github.com/weichenli0909-crypto/stock-report.git
cd stock-report
```

#### ③ 安装 Python 依赖

```bash
# 先装 Python 3.10+（https://www.python.org/downloads/）
pip3 install -r requirements.txt
```

#### ④ 配置 LLM API Key（可选，用于舆情分析）

```bash
cp .env.example .env
# 然后用记事本/VS Code 打开 .env，填上你的 API Key
```

支持的 LLM（任选一个）：
- DeepSeek：https://platform.deepseek.com
- Kimi：https://platform.moonshot.cn
- 豆包：https://www.volcengine.com/product/doubao

#### ⑤ 运行

```bash

python3 run_workflow.py

# 方式2：启动 Web 服务器（推荐，可点"刷新"按钮）
```

## 🆘 万一 GitHub 打不开怎么办？

国内偶尔访问 GitHub 慢，解决方案：

1. **用代理/VPN** — 最稳定
2. **用 Gitee 镜像** — 可以手动同步到 Gitee（https://gitee.com）做双备份
3. **改用 GitHub CDN 加速** — `git config --global url."https://ghproxy.com/https://github.com/".insteadOf "https://github.com/"`
